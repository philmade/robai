
# Examples for RobAI Framework

This directory contains various examples showcasing the capabilities and flexibility of the RobAI framework.

## List of Examples

1. **Language Translation Robot**:
   Translates a given text from one language to another using a mock AI model.
   
2. **Weather Reporting Robot**:
   Fetches the weather report for a specified location using a mock AI model.
   
3. **Crypto Price Checker Robot**:
   Retrieves the current price of a given cryptocurrency using a mock AI model.
   
4. **Web Scraper Robot**:
   Scrapes content from a provided URL using a mock AI model.

## How to Run

To execute any example, navigate to its respective Python script and run it.

```bash
python <name_of_the_script>.py
```

For a consolidated execution, run:

```bash
python sample_robots.py
```

Note: These examples use mock AI models and produce dummy results. They are designed to demonstrate the structure and potential of the RobAI framework.
